# Real Time Stock Market Dashboard

## Features
- Real-Time Stock Data
- Company Information
- Candlestick Chart
- Volume Analysis
- Moving Averages (20, 50, 200)
- RSI Indicator
- Bollinger Bands
- Buy/Sell Signal
- Portfolio Tracker
- Risk Analysis
- Price Prediction
- CSV Download

## Technologies Used
- Python
- Streamlit
- Plotly
- Pandas
- NumPy
- yfinance
- Scikit-learn
